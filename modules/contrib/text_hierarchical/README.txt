-- SUMMARY --

The Hierarchical Text module is based on core Text module. It defines three
field types: Hierarchical text, Hierarchical long text, and Hierarchical long
text and summary. Each of these fields is extended version of corresponding
fields defined by Text module with ability to be stored and displayed as
hierarchical trees.


For a full description of the module, visit the project page:
  http://drupal.org/project/text_hierarchical

To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/text_hierarchical


-- REQUIREMENTS --

None.


-- INSTALLATION --

* Install as usual, see https://drupal.org/node/895232 for further information.
